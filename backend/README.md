# ⚡ Watt-Mate (와트메이트)
> **한전 AMI 데이터를 활용한 AI 기반 지능형 에너지 관리 플랫폼**

Watt-Mate는 단순한 전력 모니터링을 넘어, **한전 EDS API**로 수집된 데이터를 AI가 분석하여 가구별 맞춤형 에너지 절약 솔루션을 제공하고, 사용자 간의 **에너지 절약 리그**를 통해 재미있게 전기 요금을 줄일 수 있도록 돕는 스마트 홈 에너지 파트너입니다.

---

## 📌 Project Overview
기존의 전력 확인 서비스들이 제공하는 단순한 수치 나열에서 벗어나, 사용자의 생활 패턴을 학습하고 **가전별 사용 추론(NILM)** 및 **누진세 예측 알림**을 통해 실질적인 비용 절감을 돕습니다.

### Key Value Proposition
* **No Hardware:** 별도의 스마트 플러그 설치 없이, 기존 한전 AMI 계량기 데이터를 활용합니다.
* **AI Insight:** 단순 통계가 아닌 딥러닝 모델 기반의 정밀한 미래 사용량 예측을 제공합니다.
* **Actionable Guide:** "절약하세요"라는 구호 대신, "지금 건조기를 끄면 누진세를 피할 수 있어요"라는 구체적 행동을 제안합니다.
* **Gamification:** 리그 시스템을 통해 사용자 간의 선의의 경쟁을 유도하고, 지속적인 절약 동기를 부여합니다.

## 🛠 Tech Stack
프로젝트는 크게 4개의 모듈로 구성됩니다.

| Module | Responsibility | Tech / Tools |
| :--- | :--- | :--- |
| **`watt-strategy`** | 요구사항 분석 및 전략 기획 | Documentation, Flowchart |
| **`watt-backend`** | 한전 API 연동 및 데이터 처리 | Python, FastAPI, MySQL |
| **`watt-ai`** | 전력 패턴 분석 및 예측 모델 | PyTorch/TensorFlow, LSTM, NILM |
| **`watt-front`** | 사용자 대시보드 및 웹 UI | React |

## 🚀 Key Features

### 1. 한전 EDS API 실시간 연동
* 사용자 동의 기반의 공식 전력 데이터 수집.
* 15분/1시간 단위의 고해상도 전력 사용량 트래킹.

### 2. AI 에너지 대시보드
* **패턴 분석:** 시간대별/요일별 주간 에너지 리포트 생성.
* **가전 추론:** 전체 사용량에서 특정 가전(에어컨, 건조기 등)의 비중을 분리 분석.
* **이상 감지:** 평소와 다른 대기 전력 급증 시 즉시 푸시 알림.

### 3. 지능형 요금 예측
* 현재 사용 속도 기반 월말 예상 요금 산출.
* 누진세 구간 진입 전 선제적 경고 시스템.

### 4. 수동 입력 폴백(Fallback) 시스템
* AMI 미설치 가구를 위한 OCR(광학 문자 인식) 기반 계량기 촬영 입력 기능.

### 5. 에너지 절약 리그 시스템 (Gamification)
* **티어/랭킹 시스템:** 전월 대비 절감률 및 동일 평수/가구원 수 대비 효율에 따라 사용자에게 랭크와 티어(Tier)를 부여합니다.
* **주간/월간 이웃 리그:** 비슷한 주거 환경의 이웃들과 절약 성과를 겨루는 그룹 매칭 시스템.
* **보상 및 챌린지:** "누진세 2단계 방어하기", "주말 대기전력 줄이기" 등 퀘스트를 통한 재미 요소 및 리워드(배지 등) 제공.
